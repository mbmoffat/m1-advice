# Page roadmap

Production plan for advice.mortgageonefinance.co.uk. This subdomain takes the question layer and amount pages only. It never targets the head terms "expat mortgages" or "seafarer mortgages" (www owns /expats and /seafarers), and never targets the items in the CLAUDE.md "Never target" list.

Non-autonomous: every page ships by one pull request per page and merges only on human approval. Build one cluster fully before starting the next. Cadence is two to four new pages per day maximum.

## Build order

Seafarer, then expat, then complex income, then later-life (55 to 70), then HNW amount pages, then first-time buyer, then buy-to-let, then equity release and lifetime mortgages last.

## Status key

- planned: agreed and on the roadmap, not yet built
- built: page exists in the repo, pull request open or merged, not yet deployed
- live: deployed and reachable on the subdomain
- indexed: confirmed indexed in Google Search Console

## Volume and difficulty

Monthly volume and difficulty are filled by the per-page protocol (Ahrefs keyword and difficulty check) at build time. Rows are seeded as TBC rather than estimated, so no unverified figure enters the plan.
