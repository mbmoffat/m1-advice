# Build queue — advice.mortgageonefinance.co.uk

Committed build queue for advice.mortgageonefinance.co.uk so scheduled and remote
runs can read it. The page builder works this list top to bottom and obeys CLAUDE.md
over anything here. Build one cluster fully before starting the next.

## Build order

Seafarer, then expat, then complex income, then later-life (55 to 70), then HNW
amount pages, then first-time buyer, then buy-to-let, then Halal Mortgages, then
equity release and lifetime mortgages last.

Halal Mortgages is a deliberately broad cluster: a larger page count and wider
keyword footprint, broad explainer pages, an intentional exception to the
high-intent default, not drift.

## Status key

- planned: agreed and on the queue, not yet built
- PR open: pull request open, auto-merges on green guards and Vercel
- live: deployed and reachable on the subdomain
